Windows 安装与激活

发表于2023-10-09|更新于2024-01-31|Windows
|字数总计:1.4k|阅读时长:4分钟|阅读量:

注意：本章所使用的方法与工具均来自互联网，并且经本人亲自测试，出现问题概不负责，仅供参考学习！

简介

  • Windows 是美国微软公司以图形用户界面为基础研发的操作系统，是全球应用最广泛的操作系统之一

系统版本

  • 所使用的系统资源与镜像写入工具都选用官方或知名软件

1.版本选择

  • 根据电脑硬件配置选择合适的系统版本
  • 其次根据需要选择系统版本功能

以下系统版本只介绍部分版本仅供参考

  • Windows XP：适合旧设备使用，功能较少但是兼容性强（停更时间2014年4月8日）
  • windows 7：新老设备都可使用，功能性稳定兼容性也强（停更时间2020年1月14日）
  • Windows 10: 目前主流系统，对旧设备兼容较差，版本功能多（持续更新中）
  • Windows 11：最新版本系统，功能丰富且稳定流畅，对旧设备兼容差（持续更新中）

2.选择推荐

  • Windows XP：老旧设备、不玩游戏、日常看剧、轻办公
  • Windows 7：日常办公、轻度游戏、轻度3D建模与剪辑等
  • Windows 10：企业办公、主流游戏、设计师与工程师
  • Windows 11：企业办公、最新游戏、最新软件使用与最新设备

3.资源下载

  • 下载 Windows 10
  • 下载 Windows 11

旧版本系统官方已经关闭下载，推荐此资源下载

  • NEXT, ITELLYOU

  • MSDN, 我告诉你

镜像写入工具推荐

  • UltraISO

  • Rufus

系统安装

  • 安装的系统为 Windows 10，材料准备一个8G 以上的U盘，且提前检查硬件是否达到 Windows 10系统安装要求，Windows 10 系统要求和规格

1.系统盘制作

  • 下载 Windows 10 镜像工具，下载 Windows 10

  • 打开镜像工具，选择接受

  • 选择创建安装介质，下一步

  • 选择下一步

  • 选择U盘，下一步

  • 等待下载完成即可，拔掉U盘，系统盘制作完成

2.安装系统

  • 根据电脑主板启动进入电脑BIOS，修改启动引导选项至U盘，软后重启系统就如下图，所有BIOS启动与按键

  • 选择我没有密钥

  • 选择自定义安装

  • 下一步，也可以选择硬盘进行分盘，给系统至少100G容量

  • 安装好后拔掉U盘，最后等待几分钟就能进入系统了

激活系统

系统激活分为一下三种方式

密钥激活：通过微软购买的密钥进行激活，在系统激活页面激活

KMS激活：企业计算机批量激活，企业内部KMS主机激活所有连接的电脑，一般是有时间限制的

OEM激活：又称数字许可证激活，是微软授权给各大电脑品牌批量激活的，只要电脑硬件不改变，重装系统后自动激活

根据需要推荐激活方式

  • 普通大众：非公司非单位电脑，仅个人电脑，推荐KMS激活和OEM激活，当然想要正版密钥激活更好
  • 企业单位：在企业或单位工作的建议购买正版密钥激活系统，否则可能会有律师函警告，带来不必要麻烦

1.密钥激活

  • 购买密钥后，将密钥序列号输入到激活页面，确定激活即可

2.KMS 激活

  • KMS激活原理就是连接KMS主机并授权激活时间即可，提供两种方法代码激活与HEU KMS Activator工具激活
  • 右键点击开始按钮，选择 Windows PowerShelll（管理员）或 终端（管理员），执行命令
  • slmgr.vbs /upk （卸载已安装密钥）
  • slmgr /ipk *** （星号代表的是密钥）
  • slmgr /skms *** （星号代表的是IP地址或网址）

演示如下

  • 卸载已安装密钥
  • 安装密钥：W269N-WFGWX-YVC9B-4J6C9-T83GX，该密钥为Windows 10 / 11 专业版通用
  • 设置 KMS 激活服务器，即可
  • 其他Windows10/11版本密钥，激活服务器IP：192.168.7.186

演示结果

3.OEM 激活

  • 使用HEU KMS Activator默认密码：1234工具里的OEM激活功能即可，前提是登录微软账号

问题疑问

  • 启动盘写入没有数据或报错，检查U盘有没有加密、上锁、损坏，格式选择NTFS
  • 系统安装时出现未知报错，检查电脑是否符合该系统与更换其他U盘，检查BIOS磁盘分区是否为GPT分区
  • 系统WiFi、扬声器、麦克风、键盘、鼠标等无法使用，下载好所有硬件驱动使用PE系统进行驱动安装
  • 系统安装好后频繁出现黑屏几秒或闪屏，这是系统正在更新硬件驱动导致，不是问题，更新完即可恢复
  • 桌面没有此电脑图标，先将系统激活，右键选择个性化👉主题👉桌面图标设置👉勾选此电脑👉应用确定即可
  • HEU KMS Activator默认密码：1234工具打开时会报毒，请将系统安全里的实时保护暂时关闭，激活完系统后再打开
