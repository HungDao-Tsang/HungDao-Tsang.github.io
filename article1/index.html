з®ҖеҚ•д»Ӣз»Қ

  • Hexo

    HexoжҳҜдёҖдёӘеҝ«йҖҹгҖҒз®ҖжҙҒгҖҒй«ҳж•ҲгҖҒејҖжәҗгҖҒжңүеӨҡз§Қдё»йўҳзҡ„еҚҡе®ўжЎҶжһ¶пјҢHexoдҪҝз”ЁMarkdownжҲ–иҖ…е…¶д»–жёІжҹ“еј•ж“Һи§Јжһҗж–Үз« пјҢеңЁеҮ з§’й’ҹеҶ…еҚіеҸҜз”ҹжҲҗдёҺдё»йўҳдёҖж ·йқ“дёҪзҡ„йқҷжҖҒзҪ‘йЎө

  • Next

    Nextдё»йўҳжҳҜHexoзҡ„ејҖжәҗдё»йўҳпјҢзӣ®еүҚд№ҹжҳҜдҪҝз”ЁжңҖеӨҡзҡ„дё»йўҳд№ӢдёҖпјҢе…¶йЈҺж јжңҖдёәжһҒз®ҖпјҢдё»йўҳеҶ…жңүеӣӣз§ҚеёғеұҖеҸҜйҖүпјҢжң¬еҚҡе®ўз”Ёзҡ„е°ұжҳҜNextдё»йўҳзҡ„GeminiеёғеұҖ

жҗӯе»әзҺҜеўғ

е®үиЈ…Node.js

  • Node.jsе®үиЈ…ж–№жі•зҪ‘дёҠжңүеҫҲеӨҡиҖҢдё”еҫҲиҜҰз»ҶпјҢжң¬ж–ҮдёҚи®°еҪ•пјҢеӨ§жҰӮжӯҘйӘӨе®ҳзҪ‘дёӢиҪҪжңҖж–°зүҲпјҢдёҖзӣҙдёӢдёҖжӯҘе®үиЈ…пјҢдёҚжӣҙж”№д»»дҪ•и®ҫзҪ®дёҺе®үиЈ…дҪҚзҪ®пјҢHexoиҝҗиЎҢеҝ…йЎ»з”ЁеҲ°Node.jsзҺҜеўғ

е®үиЈ…Git

  • Gitе®үиЈ…ж–№жі•зҪ‘дёҠд№ҹеҫҲеӨҡеҫҲиҜҰз»ҶпјҢжң¬ж–ҮдёҚи®°еҪ•пјҢе®ҳзҪ‘дёӢиҪҪеҫҲж…ўеҸҜиғҪиҝҳи®ҝй—®дёҚдәҶпјҢжҺЁиҚҗй•ңеғҸжәҗдёӢиҪҪпјҢе®үиЈ…ж–№жі•д№ҹжҳҜдёӢиҪҪжңҖж–°зүҲпјҢдёҖзӣҙдёӢдёҖжӯҘе®үиЈ…пјҢдёҚжӣҙж”№д»»дҪ•и®ҫзҪ®е’Ңе®үиЈ…дҪҚзҪ®пјҢGitжҳҜдёҖдёӘеҲҶеёғејҸйғЁзҪІзҡ„е·Ҙе…·пјҢеҗҺйқўHexoе®үиЈ…гҖҒжҸ’д»¶е®үиЈ…гҖҒNextдё»йўҳдёӢиҪҪгҖҒеҸ‘еёғж–Үз« гҖҒдёҠдј GitHubзӯүзӯүйғҪйңҖGit

жҗӯе»әдјҳеҢ–

зј–иҫ‘еҷЁе®үиЈ…

  • Visual Studio Code
  • Visual Studio Codeзј–иҫ‘еҷЁеҠҹиғҪејәеӨ§пјҢдҪ“з§Ҝе°ҸпјҢжӢҘжңүеҗ„з§Қзј–зЁӢиҜӯиЁҖжҸ’д»¶пјҢзј–иҫ‘еҷЁйңҖиҰҒз»Ҹеёёз”ЁеҲ°пјҢе®үиЈ…ж–№жі•зӣҙжҺҘдёӢиҪҪе®үиЈ…

е®үиЈ…ж·ҳе®қй•ңеғҸжәҗ

  • CMDжү§иЎҢд»ҘдёӢе‘Ҫд»Ө
  • hexoеҚҡе®ўжЎҶжһ¶пјҢNextдё»йўҳпјҢеҗ„з§ҚжҸ’д»¶еӨ§йғЁеҲҶйғҪеҸ‘еёғеңЁGitHubпјҢи®ҝй—®дёӢиҪҪзү№еҲ«ж…ўиҝҳеҸҜиғҪеӨұиҙҘпјҢжүҖд»ҘжӣҙжҚўеӣҪеҶ…зҡ„ж·ҳе®қй•ңеғҸжәҗдёӢиҪҪпјҢжӣҙж”№д№ӢеҗҺйңҖиҰҒжіЁж„Ҹв¬ҮпёҸ

Gitз»‘е®ҡ

  • GitHub

GitHubжіЁеҶҢ

  • GitHubиҙҰеҸ·жіЁеҶҢйқһеёёз®ҖеҚ•пјҢеҰӮжһңи®ҝй—®GitHubеӨұиҙҘпјҢHong Daoдё»йЎөзҡ„зҙўеј•зҪ‘з»ңе·Ҙе…·йҮҢдёӢиҪҪGitHubи®ҝй—®е·Ҙе…·

з”ҹжҲҗе…¬й’Ҙ

  • CMDжү§иЎҢд»ҘдёӢе‘Ҫд»ӨпјҢз”ҹжҲҗssh key
  • еҜҶй’Ҙз”ҹжҲҗзҡ„и·Ҝеҫ„дёә

GitHubеЎ«еҶҷеҜҶй’Ҙ

  • и®°дәӢжң¬жҲ–иҖ…зј–иҫ‘еҷЁжү“ејҖз”ҹжҲҗзҡ„е…¬й’ҘпјҢеӨҚеҲ¶йҮҢйқўжүҖжңүеҶ…е®№

  • зҷ»еҪ•GitHubпјҢзӮ№еҮ»еӨҙеғҸвһЎпёҸSettingsвһЎпёҸSSH and GPG keysвһЎпёҸNew SSH keyвһЎпёҸжҠҠе…¬й’ҘеҶ…е®№зІҳиҙҙиҝӣеҺ»вһЎпёҸзӮ№еҮ»Add SSH Key

  • CMDжү§иЎҢд»ҘдёӢе‘Ҫд»ӨпјҢжҹҘзңӢжҳҜеҗҰз»‘е®ҡ
  • иӢҘеӣһеӨҚиҫ“еҮәд»ҘдҝЎжҒҜдёӢеҚідёәз»‘е®ҡжҲҗеҠҹ

еҲӣе»әеҚҡе®ўд»“еә“

  • зҷ»еҪ•GitHubвһЎпёҸйҰ–йЎөзӮ№еҮ»cerate a new repository еҲӣе»әд»“еә“вһЎпёҸи®ҫзҪ®Repository nameд»“еә“еҗҚз§°
  • д»“еә“еҗҚз§°ж јејҸпјҡGitHubз”ЁжҲ·еҗҚ.github.io
  • дҫӢеҰӮпјҡdao943.github.io
  • еҝ…йЎ»йҖүжӢ©PublicејҖжәҗвһЎпёҸзӮ№еҮ»Creating repositoryе®ҢжҲҗеҲӣе»ә

  • з»§з»ӯжү§иЎҢд»ҘдёӢе‘Ҫд»ӨпјҢжҹҘзңӢжҳҜеҗҰе®үиЈ…жҲҗеҠҹпјҢеҮәзҺ°зүҲжң¬еҸ·еҚідёәе®үиЈ…жҲҗеҠҹ

еҲқе§ӢеҢ–еҚҡе®ў

  • еҲӣе»әеӯҳж”ҫеҚҡе®ўж–Үд»¶еӨ№вһЎпёҸжү“ејҖж–Үд»¶еӨ№вһЎпёҸеҸій”®Git Bash Hereжү“ејҖGit

  • жү§иЎҢеҲқе§ӢеҢ–е‘Ҫд»ӨпјҲGitдёҚиғҪдҪҝз”Ёеҝ«жҚ·й”®еӨҚеҲ¶дёҺзІҳиҙҙпјҢе…·дҪ“зҷҫеәҰпјү

  • еҰӮжһңеҮәзҺ°жҠҘй”ҷпјҢеүҚйқўжӯҘйӘӨжІЎжңүд»»дҪ•ж”№еҠЁжҲ–иҖ…еҒҡй”ҷпјҢйӮЈеӨ§жҰӮзҺҮжҳҜзҪ‘з»ңй—®йўҳпјҢи§ЈеҶіеҠһжі•еӨҡиҜ•еҮ ж¬ЎжҲ–иҖ…з§‘еӯҰдёҠзҪ‘пјҢеҰӮиҝҳжҳҜдёҚиЎҢиҜ·зҷҫеәҰ

  • еӣһеӨҚиҫ“еҮәд»ҘдёӢдҝЎжҒҜдёәеҲқе§ӢеҢ–жҲҗеҠҹ
